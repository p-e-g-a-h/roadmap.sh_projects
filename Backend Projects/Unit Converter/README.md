# Web Unit Converter

Project Challenge: [roadmap.sh - Unit Converter](https://roadmap.sh/projects/unit-converter)

A lightweight, full-stack application built with native Node.js (no frameworks) to handle unit conversions and serve static assets.

## Installation

1. **Clone the repository:**

```bash
git clone https://github.com/p-e-g-a-h/roadmap.sh_projects.git
```

2. **Navigate to the project folder:**

```bash
cd "roadmap.sh_projects/Backend Projects/Unit Converter"
```

3. **Install dependencies:**

```bash
npm install
```

## Usage

1. **Start the server:**

```bash
node index.js
```

2. **Open your browser:** Go to `http://localhost:3000`
